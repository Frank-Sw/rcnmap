#rcnmap

draw the province map with specified color in China map.

##install
```
library(devtools)
install_git("https://github.com/jiang-hang/rcnmap")
```

##usage
```
library(rcnmap)
cnmap(c("hubei","shanghai","gansu"),c("red","blue","yellow"))
```
